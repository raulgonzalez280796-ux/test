# Directory Size Analyzer

A simple CLI tool written in Python to calculate and display the total size of a given directory.

## Usage

Run the analyzer via command line:
`python analyzer.py -p /path/to/directory`

Run tests using pytest:
`pytest test_analyzer.py`
